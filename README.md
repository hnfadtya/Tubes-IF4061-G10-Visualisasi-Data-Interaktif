# Tubes-IF4061-G10-Visualisasi-Data-Interaktif

## SawitHub

SawitHub is an interactive data visualization dashboard that provides insights and information about the Indonesian palm oil industry. Built using **Dash (Plotly Dash)**, the platform enables users to explore data through interactive charts and visual analytics

## Live Dashboard

The dashboard is available online at:

https://sawithub.onrender.com/

## Features

- Interactive data visualizations
- Dashboard-based analytics
- Insights into the Indonesian palm oil industry
- Responsive and user-friendly interface
- Built with Plotly Dash

## Tech Stack

- Python
- Dash (Plotly Dash)
- Pandas
- Plotly

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/Tubes-IF4061-G10-Visualisasi-Data-Interaktif.git
cd Tubes-IF4061-G10-Visualisasi-Data-Interaktif
```

2. Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Running Locally

Run the Dash application:

```bash
python app.py
```

After the application starts, open your browser and navigate to:

```text
http://127.0.0.1:8050/
```
